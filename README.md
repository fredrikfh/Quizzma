# 🧑‍🏫 Quizzma 🧑‍🎓

Student response system leveraging natural language processing (NLP) techniques to manage open-text responses in live (synchronous) lecture settings.

Quizzma was created by **Fredrik Fonn Hansen** and **Sander Østrem Fagernes** at the **Norwegian University of Science and Technology (NTNU)** as part of the master's thesis

>`"Applying AI techniques for synchronous analysis of students' open-text responses in higher education classrooms"`

## 🛜 Production instance

For a limited time, a running production instance will be available at:

> [https://quizzma.no](https://quizzma.no) - Main page for host/teacher. Register user, log in, or see all quizzes.

> [https://quizzma.no/join](https://quizzma.no/join) - Page for audience/student. Join a quiz session by entering a 4-digit pin.

> [https://backend.quizzma.no/docs](https://backend.quizzma.no/docs) - Documentation of the backend API generated by OpenAPI.
